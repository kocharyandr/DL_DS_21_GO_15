# Глубинное обучение 

Материалы по курсу "Глубинное обучение" потока DS-16 дополнительного профессионального образования от ФКН НИУ ВШЭ.

## Общая информация
1. В курсе предусмотрено **3 домашних задания** по **10 баллов** за каждое.
2. Формула оценки (округление арифметическое):\
$$S = \text{round}\left(\frac{1}{3}\sum_{i=1}^{3}HW_i\right).$$
4. Зачет: $S \geq 6$.
